## 🎯 How to Use

### For Users
1. Open app → Click Lectures
2. Click **"+ Add Video"** button
3. Select video file (.mp4)
4. Fill title, subject, duration
5. Click **"✅ Save Lecture"**
6. Click **"Watch"** to play

### For Developers
- Edit `lectures.js` to customize features
- Modify upload form in `lectures.html`
- Change storage method in `localStorage` section
- CSS styling in `style.css`

## ⚙️ Configuration

### To Adjust Max File Size
Edit `lectures.js`, find:
```javascript
if (!file.type.includes('video')) {
    alert('❌ Please select a valid video file');
    return;
}
// Add file size check here
```

### To Add More File Formats
Edit `lectures.html`, find:
```html
<input type="file" id="videoFileInput" accept="video/*">
<!-- Currently accepts all video types -->
<!-- Change accept="video/*" to specific types if needed -->
```

### To Customize Form Fields
Edit both `lectures.html` and `lectures.js` to add/remove fields

## 🐛 Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Video won't upload | Check file is .mp4 format |
| Upload hangs | Video too large, compress first |
| Video not showing after refresh | Clear browser cache, check localStorage |
| Playback stutters | Video file too compressed, increase quality |
| Can't find + Add Video button | Check page loaded completely, refresh |
