Step 2: Edit data.js

{ id: 1, title: "Your Title", youtubeId: "dQw4w9...", duration: "45 min" }

Step 3: Refresh

Reload lectures.html - video appears!

✏️ Add Tests

Edit data.js:

{ id: 1, title: "Quiz", duration: 30, questions: [{ id: 1, text: "Q?", options: ["A","B","C","D"], correctAnswer: 1 }] }

Notes:

  • correctAnswer: 0=A, 1=B, 2=C, 3=D
  • Duration in minutes
  • Exactly 4 options

⚠️ Common Mistakes

❌ Wrong

youtubeId: "https://youtube.com..." youtubeId: "dQw4w9WgXcQ " (space) youtubeId: dQw4w9WgXcQ (no quotes)

✅ Correct

youtubeId: "dQw4w9WgXcQ"

Other Common Issues

  • Missing commas between objects
  • Wrong answer index (0-3)
  • Less than 4 options in test
  • Extra spaces in YouTube ID

🔍 Troubleshooting

Lectures Not Showing?

  • Open debug.html
  • Check console (F12)
  • YouTube ID must be 11 chars
  • No spaces or special chars

Can't Login?

  • Check credentials above
  • Open console (F12)
  • Hard refresh (Ctrl+Shift+R)

Video Won't Play?

  • YouTube ID correct?
  • Video publicly available?
  • Check internet
