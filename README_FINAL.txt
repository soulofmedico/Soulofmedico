                        💾 HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════════

   UPLOAD PROCESS
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   User selects .mp4 file
            ↓
   Browser reads file as Base64
            ↓
   Save to browser localStorage
            ↓
   Video list updates
            ↓
   Students can watch immediately


   PLAYBACK PROCESS
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   Student clicks "Watch"
            ↓
   Load video from localStorage
            ↓
   Create HTML5 video element
            ↓
   Show with playback controls
            ↓
   Can adjust speed, go full-screen
