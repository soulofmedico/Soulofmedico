## ⚡ Quick Reference Commands

### JavaScript Console Commands

#### Export Backup
```javascript
const backup = localStorage.getItem('lectures');
console.log(backup);
// Copy output and save to file
```

#### Delete Specific Lecture
```javascript
const lecs = JSON.parse(localStorage.getItem('lectures'));
const filtered = lecs.filter(l => l.id !== 1);
localStorage.setItem('lectures', JSON.stringify(filtered));
location.reload();
```

#### Clear All Lectures
```javascript
localStorage.removeItem('lectures');
location.reload();
```
