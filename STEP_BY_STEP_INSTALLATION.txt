📱 ANDROID INSTALLATION (Google Chrome)

SETUP:
1. Open file on your Android phone (email it, use cloud drive, or USB transfer)
2. Open in Google Chrome (MUST be Chrome, not other browsers)

INSTALLATION METHOD 1 (Automatic Prompt):
1. After page fully loads (5-10 seconds)
2. At the bottom right, you'll see a white popup with "Install app"
3. Tap "Install"
4. Confirm any permission dialogs
5. App appears on home screen!

INSTALLATION METHOD 2 (Manual Using Menu):
1. App has fully loaded
2. Tap the 3-dot menu (⋮) at the TOP RIGHT of Chrome
3. Look for "Install app" or "Add to Home Screen"
4. Tap it
5. You may see an "Install" button at the bottom
6. Tap "Install"
7. App will be created!

INSTALLATION METHOD 3 (Using Address Bar):
1. After page loads completely
2. Look at the address bar (top of browser)
3. You might see a download icon (⬇️) or house icon
4. Tap the icon if it appears
5. Select "Install" from the popup
6. Confirm installation

WHAT HAPPENS AFTER INSTALLATION:
✓ Chrome closes
✓ App opens in full-screen mode
✓ App appears on your home screen
✓ You can pin it for quick access
✓ It works like a regular app

TROUBLESHOOTING - "Install button not showing":
✗ Chrome must be fully updated
✗ Wait 10 seconds after page loads
✗ Try refreshing the page (swipe down)
✗ Make sure JavaScript is enabled
✗ Try using the menu (⋮) method instead

Q: The app isn't installing - what do I do?
A: Try these steps:
   1. Make sure you're using the correct browser (Chrome, Safari, Edge)
   2. Wait 10 seconds after page loads
   3. Try refreshing the page
   4. Clear browser cache
   5. Try the menu (⋮) method instead of automatic button
   6. Update your browser to latest version

Q: Where's the install button?
A: It usually appears automatically when page fully loads. If not:
   1. Try the menu (⋮) and select "Install app"
   2. For iOS: Use Share → Add to Home Screen
   3. Make sure JavaScript is enabled

Q: Does it work offline?
A: Yes! After opening once while online, it works fully offline.
   Make sure to load at least one lecture while online first.

Q: Can I use it on multiple devices?
A: Yes! Each device gets its own copy with its own videos.

Q: How do I delete the app?
A: Same way as any app:
   ├─ Android: Long-press → Uninstall
   ├─ iOS: Long-press → Remove
   ├─ Windows: Settings → Apps → Uninstall
   └─ Mac: Drag to Trash or use AppCleaner

Q: Can I share videos?
A: Only if the other person has access to your device.
   Videos are stored locally, not in cloud.

Q: I forgot the password - what now?
A: The login is stored in the app code. You can:
   1. Edit login.js file
   2. Change the credentials
   3. Or contact developer for reset

Q: How much storage does it use?
A: App itself: ~5-10MB
   Each video: Size of the file (up to 500MB per upload)
   Total capacity: Depends on device (50-500MB typical)
