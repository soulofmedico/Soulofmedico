## 💾 How Videos Are Stored

### Storage Method
- **Location:** Browser's localStorage
- **Format:** Base64 encoded MP4
- **Capacity:** ~5-10MB (can vary by browser)
- **Persistence:** Survives browser restart
- **Privacy:** Completely local, no upload to cloud

### File Size Estimates
```
Original MP4:  100MB  →  Encoded: ~133MB in localStorage
Original MP4:  200MB  →  Encoded: ~266MB in localStorage
```

### Recommendation
- Keep videos under 200MB for best performance
- Compress before uploading if needed (use HandBrake)
- Test playback after uploading

## 🆘 Common Issues & Solutions

### Issue: "Can't find Admin Panel"
**Solution:**
1. Refresh page (Ctrl+F5)
2. Clear browser cache
3. Make sure you're logged in
4. Try different browser

### Issue: "Video upload fails"
**Solution:**
1. Check file is .mp4 format
2. Check file size < 500MB
3. Fill all required fields
4. Try smaller video
5. Try different browser

### Issue: "Video doesn't play"
**Solution:**
1. Try different video file
2. Check MP4 format specifically
3. Refresh page
4. Clear browser cache
5. Check browser console (F12)

### Issue: "Videos disappeared after refresh"
**Solution:**
1. Browser cache was cleared
2. localStorage was reset
3. Use export/backup feature next time
4. Check DevTools → Application → localStorage

