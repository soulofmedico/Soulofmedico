## 💾 Storage Details

### How It Works
```
1. User uploads .mp4 file
   ↓
2. Browser reads file as Base64
   ↓
3. Encoded data saved to localStorage
   ↓
4. Persists across browser restarts
   ↓
5. Auto-loads in Lectures page
```

### Storage Capacity
```
Browser localStorage: ~5-10MB per domain
Video size: 100MB → Encodes to ~133MB
Result: Can store 1-2 videos typically

For more videos:
- Compress using HandBrake
- Use web server (remove Base64 limit)
- Upload to cloud storage
```
