Issue: "Install button not showing"
Solution: 
- Wait 5-10 seconds for page to fully load
- Ensure you're using Chrome, Edge, or Safari
- Try refreshing the page (F5)
- Check if JavaScript is enabled

Issue: "App won't open after installation"
Solution:
- Try uninstalling and reinstalling
- Clear app data in settings
- Update your browser
- Restart your device

Issue: "Videos not showing in Admin Panel"
Solution:
- Ensure videos are in MP4 format
- File size must be under 500MB
- Check browser storage limits
- Clear cache if storage is full

Issue: "Offline mode not working"
Solution:
- Load app once while online to cache everything
- Service worker must register (check dev tools)
- Wait 2-3 seconds after first load
- Try going offline and refreshing

Issue: "Can't upload large videos"
Solution:
- Split videos into smaller files (<500MB)
- Use online video compressor if needed
- Check device storage space
- Clear app cache to free space
