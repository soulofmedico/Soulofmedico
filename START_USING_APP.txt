## 💾 Storage System

### How It Works
```
1. Upload .mp4 file
   ↓
2. Browser reads as Base64
   ↓
3. Save to localStorage
   ↓
4. Persist in browser
   ↓
5. Play anytime, offline
```

### Capacity
```
Storage: ~5-10MB per browser
Size: 100MB video → 133MB encoded
Tip: Keep videos under 200MB
```
