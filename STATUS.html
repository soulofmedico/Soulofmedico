🎥 How to Add Videos

  1. Get YouTube Video ID

    From URL: https://www.youtube.com/watch?v=dQw4w9WgXcQ

    Extract ID: dQw4w9WgXcQ (11 characters)

  2. Edit data.js file

    Add to the lectures array:

    { id: 2, title: "Your Lecture Title", subject: "Chemistry", youtubeId: "dQw4w9WgXcQ", duration: "45 min", description: "Full description here", notes: "Important notes" }
  3. Refresh the page

    Press Ctrl+R or Cmd+R in browser

📝 How to Add Tests

Edit data.js and add to the tests array:

{ id: 1, title: "Chemistry Quiz", description: "Test your knowledge", duration: 30, questions: [ { id: 1, text: "What is the SI unit of current?", options: ["Ohm", "Ampere", "Volt", "Watt"], correctAnswer: 1, explanation: "Ampere is the SI unit" } ] }

Important:

  • correctAnswer is the index (0, 1, 2, or 3)
  • Must have exactly 4 options
  • Duration is in minutes

🔧 Troubleshooting

❓ Video not loading?

Solution:

  • Check YouTube ID is exactly 11 characters
  • Verify video is public (not private/unlisted)
  • Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
  • Open debug.html to test video embedding

❓ Lectures not showing?

Solution:

  • Check data.js has valid JSON syntax
  • Use commas between objects in array
  • No trailing commas in last item
  • Check browser console (F12) for errors

❓ Login not working?

Solution:

  • Use exact test credentials provided
  • Email: student@example.com
  • Password: 123456
  • Check if cookies/storage are enabled
